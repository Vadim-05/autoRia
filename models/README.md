# Тренування моделі 
https://spacy.io/usage/training

Створити конфіг 
```bash
python -m spacy init fill-config base_config.cfg config.cfg
```

Тренування моделі
```djangourlpath
python -m spacy train config.cfg --output ./output --paths.train ../datasets/train.spacy --paths.dev ../datasets/train.spacy
```
